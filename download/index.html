A Few Notes Linux Users


To run Glyph, you need to change its permissions. This process is different depending on what distribution you're running, but you need to allow the file "Glyph" to be run as an executable. On Ubuntu 12.10, right click the file, click "Properties", and under the "Permissions" tab, tick "Enable executing this file as an application." Then click "OK" to exit the dialogue.

Another quirk on Linux is that there isn't a screen resolution dialogue like Windows and Max OS X. To change your screen resolution, run Glyph, close it, and edit the configuration file in ~/.config/unity3d/Brozingaron/Glyph/

I'm sorry for that inconvinience, but don't worry, Alpha 2 will have the settings menu in-game.
