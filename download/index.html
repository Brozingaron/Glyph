Alpha 2 Release Notes

  • Added energy fountains
    • hover over them to speed up your recharge rate by a factor of 2.
  • Changed recharge rates
    • 5, 15, 25, and 45 seconds from the previous bar respectively.
  • Gameplay progression
  • Added music
  • Added splash text
  • Most menus are navigable with the keyboard
  • New settings menu
  • New graphics options
  • Removed startup settings screen
  • Removed FPS meter
Known Bugs:
  • With a lot of enemies on the screen, framerates drop dramatically due to path calculations
    • The issue is not render time, but CPU calculation time. With 200+ enemies, the main thread can take between 3 and 5 seconds opposed to a preferred 15-25 ms.
  • Insane Mode is not a bug.
See the blog for more details

Alpha 1 Release Notes


Alpha 1 is the initial release of Glyph. Arcade mode is the only gameplay style currently available, but other modes and new abilities and gameplay mechanics will be added in future updates.

A Few Notes for Linux Users



To run Glyph, you need to change its permissions. This process is different depending on what distribution you're running, but you need to allow the file "Glyph" to be run as a program. On Ubuntu 12.10, right click the file, click "Properties", and under the "Permissions" tab, tick "Allow executing file as program." Then click "OK" to exit the dialogue.
