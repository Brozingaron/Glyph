Nightly Notes


Nightly builds are mid-milestone builds of Glyph. They may be buggy, slow, unstable or have halfway implemented features. All content in nightly builds can change over the course of the build cycle. The advantage to nightly builds is that they may contain pre-release features or contain more content than the current version.

Nightly builds are compiled only for Windows, and source code for Unity. Although Brozingaron does not provide compiled nightly builds for other operating systems, the code can easily be compiled for them with Unity for Windows or Mac OS X.

If you're not sure what you should do or how you got to this page, we recommend you see the standard download page.
