Nightly Notes


Nightly builds are mid-milestone builds of Glyph. They may be buggy, slow, unstable or have halfway implemented features. All content in nightly builds can change over the course of the build cycle. The advantage to nightly builds is that they may contain pre-release features or contain more content than the current version.

Nightly builds are not compiled for all platforms, but the source code is provided for Unity which can be compiled into binaries for other operating systems.

IMPORTANT NOTICE: Using a nightly build of Glyph may erase all game progress on both newer and older milestone builds.

If you're not sure what you should do or how you got to this page, please see the standard download page.

Additional release notes can be found on SourceForge.
