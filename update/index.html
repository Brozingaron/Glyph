An Update to Glyph is Available

January 31, 2014


Alpha 3



Choose your operating system





All downloads are .zip files for x86 systems
The source code download above is for Unity.
Source cannot be natively played on any platform without being compiled.

Alpha 3 Release Notes

  • Added boss!
    • The Trapezoid of Doom is now in the game
  • Added lives to arcade and insane mode
  • New music for insane mode
  • New enemy sprite
  • New pause sounds
  • Other minor gameplay improvements
  • Changed recharge rates
    • Fountains now reduce the recharge time by a factor of 4
  • Gameplay progression
  • New/modified splash text
  • New controls menu
  • Take a screenshot with F4
  • Added a very basic update checker which launches the browser if an update is found
  • Reduced game size dramatically by removing psd's and other source files before compiling
  • Added game console
    • Once enabled in the controls settings, press ` (the accent key, normally on the same button as ~) to open the console.
  • Note that this update will erase all scores and settings from other versions of Glyph
Known Bugs:
  • The number of enemies is limited to 50, at which point a boss spawns
    • The issue is not a bug, but it affects gameplay. The motion script needs to be rewritten to be more efficient.
  • When enemies transform into bosses, it often produces a huge lag spike
